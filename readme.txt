A simple Elementor addon for creating beautiful gradient buttons with "Gleam" hover effects.

== Description ==

Gleam for Elementor is a straight forward elementor addon that adds a new Elementor widget to Elementor. Using this so called "Gleam Button" widget, you'll be able to easily create a clean gradient button with gleam hover effects!

Features:

*   Easy to use
*   Elementor compatible
*   Open-source
*   Responsive
